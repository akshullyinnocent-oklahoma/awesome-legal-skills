# Instructions d'utilisation - Fichier Kahoot!

Votre QCM léger a été généré au format Kahoot!. Voici la procédure complète pour l'importer dans votre compte Kahoot!.

## Caractéristiques du fichier généré

- **Format** : Fichier Excel (.xlsx)
- **Type de QCM** : Léger (questions et réponses courtes, optimisées pour usage ludique)
- **Compatibilité** : Toutes les questions respectent les contraintes Kahoot! (95/60 caractères)
- **Randomisation** : Les réponses sont déjà dans un ordre aléatoire

## Procédure d'importation dans Kahoot!

### Étape 1 : Accéder au créateur Kahoot!

1. Connectez-vous à votre compte Kahoot! sur [https://kahoot.com](https://kahoot.com)
2. Cliquez sur le bouton **Create** en haut à droite de l'écran
3. Sélectionnez le type **Kahoot** (quiz interactif)

### Étape 2 : Configurer les informations de base

1. Donnez un **titre** à votre Kahoot
2. Ajoutez une **description** (optionnel)
3. Ajoutez une **image de couverture** (optionnel mais recommandé pour l'engagement)
4. Sélectionnez la **visibilité** (privé, partage avec des collègues, ou public)

### Étape 3 : Importer les questions

1. Dans le créateur, cherchez le bouton **Add question** dans le panneau de droite
2. Cliquez sur la **petite flèche** à côté de "Add question"
3. Sélectionnez **Import spreadsheet** dans le menu déroulant
4. Une fenêtre pop-up s'ouvre

### Étape 4 : Charger le fichier

1. Cliquez sur **Choose file** ou **glissez-déposez** le fichier .xlsx généré
2. Cliquez sur **Upload**
3. Kahoot! vérifie automatiquement le fichier pour détecter d'éventuelles erreurs

### Étape 5 : Vérification et correction

1. **Si aucune erreur** : Toutes les questions s'affichent avec une coche verte ✓
   - Cliquez sur **Continue** pour finaliser l'import
   
2. **Si des erreurs sont détectées** : Un message détaille les problèmes
   - **Option A** : Cliquez sur **Go back** pour corriger le fichier Excel et recommencer
   - **Option B** : Cliquez sur **Continue without errors** pour importer uniquement les questions valides (les autres sont ignorées)

### Étape 6 : Personnalisation post-import

Une fois les questions importées, vous pouvez les personnaliser davantage :

#### Ajout d'images
- Cliquez sur une question
- Cliquez sur **Add image** ou l'icône d'image
- Uploadez une image depuis votre ordinateur ou cherchez dans la bibliothèque Kahoot!

#### Ajout de vidéos
- Cliquez sur une question
- Cliquez sur **Add video**
- Intégrez une vidéo YouTube en collant l'URL

#### Ajustement du temps
- Par défaut : 20 secondes par question
- Cliquez sur l'horloge pour modifier (5, 10, 20, 30, 60, 90, 120, ou 240 secondes)
- Adaptez selon la difficulté de chaque question

#### Modification des réponses
- Vous pouvez modifier le texte de n'importe quelle réponse
- Vous pouvez changer quelle(s) réponse(s) est/sont correcte(s)
- **Attention** : Ne pas dépasser 60 caractères par réponse

#### Réorganisation des questions
- Glissez-déposez les questions pour changer leur ordre
- Par défaut, elles suivent l'ordre du fichier Excel (déjà organisé par difficulté croissante)

### Étape 7 : Configuration des paramètres du jeu

Avant de lancer le Kahoot, configurez les paramètres :

1. Cliquez sur l'icône **Settings** (roue dentée)
2. Choisissez les options :
   - **Question order** : Séquentiel (recommandé) ou aléatoire
   - **Answer options** : Mélanger les réponses pour chaque joueur (recommandé)
   - **Points system** : Standard (0-1000 points selon rapidité) ou personnalisé
   - **Affichage des réponses** : Après chaque question ou à la fin

### Étape 8 : Lancement du Kahoot

1. Cliquez sur **Play** en haut à droite
2. Choisissez le mode :
   - **Teach** : Mode synchrone en classe (tous jouent en même temps)
   - **Assign** : Mode asynchrone (les étudiants jouent à leur rythme)
   - **Challenge** : Envoi individuel avec lien unique

3. Pour le mode **Teach** :
   - Un **PIN de jeu** s'affiche à l'écran
   - Les étudiants vont sur [kahoot.it](https://kahoot.it) et entrent le PIN
   - Vous contrôlez le rythme du jeu depuis votre écran

4. Pour le mode **Assign** :
   - Définissez une date limite
   - Copiez le lien à envoyer aux étudiants
   - Ils jouent individuellement quand ils le souhaitent

## Résolution de problèmes courants

### Erreur : "Question too long"

**Cause** : Une question dépasse 95 caractères (normalement impossible si fichier bien généré)

**Solution** : Éditez manuellement la question dans Kahoot! ou réimportez après correction du fichier

### Erreur : "Answer too long"

**Cause** : Une réponse dépasse 60 caractères (normalement impossible si fichier bien généré)

**Solution** : Raccourcissez la réponse directement dans Kahoot! ou dans le fichier Excel

### Erreur : "Invalid time limit"

**Cause** : Valeur de temps non autorisée

**Solution** : Les valeurs doivent être : 5, 10, 20, 30, 60, 90, 120, ou 240 secondes

### Les caractères accentués s'affichent mal

**Cause** : Problème d'encodage (rare avec .xlsx)

**Solution** : Réenregistrer le fichier Excel en s'assurant qu'il est bien en UTF-8

### Certaines questions ne s'importent pas

**Cause** : Erreur de format dans le fichier Excel

**Solution** : 
1. Vérifier que la première ligne contient bien les en-têtes exacts
2. Vérifier que chaque question a au moins 2 réponses
3. Vérifier que "Correct answer(s)" contient des numéros valides (1-4) sans espaces

## Conseils pour un Kahoot! engageant

### Avant le jeu

1. **Testez votre Kahoot** : Jouez-le une fois en mode "Preview" pour vérifier que tout fonctionne
2. **Préparez les images** : Un Kahoot avec images est beaucoup plus engageant
3. **Ajustez les temps** : 20 secondes pour questions factuelles, 30+ pour questions nécessitant réflexion

### Pendant le jeu

1. **Commentez les réponses** : Après chaque question, expliquez brièvement pourquoi telle réponse est correcte
2. **Encouragez la participation** : Félicitez l'engagement, pas seulement les bonnes réponses
3. **Gérez le rythme** : Ne laissez pas trop de temps mort entre questions

### Après le jeu

1. **Téléchargez le rapport** : Kahoot! génère un fichier Excel avec les résultats détaillés
2. **Analysez les questions problématiques** : Si <50% de bonnes réponses, la question était peut-être mal formulée
3. **Partagez les résultats** : Option de partager le podium final avec les étudiants

## Exportation des résultats

1. Après avoir joué, allez dans **Reports**
2. Sélectionnez la session jouée
3. Cliquez sur **Download results** (Excel)
4. Le fichier contient :
   - Résultats par joueur
   - Résultats par question
   - Temps de réponse moyens
   - Statistiques globales

## Réutilisation et partage

### Dupliquer le Kahoot

1. Dans votre bibliothèque, trouvez le Kahoot
2. Cliquez sur les **trois points** (...)
3. Sélectionnez **Duplicate**
4. Modifiez la copie sans affecter l'original

### Partager avec des collègues

1. Ouvrez le Kahoot
2. Cliquez sur **Share** (icône de partage)
3. Choisissez l'option :
   - **Copy link** : Lien direct
   - **Email** : Envoi par email
   - **QR code** : Génération d'un QR code

### Exporter vers un autre format

Kahoot! ne permet pas d'exporter directement, mais vous pouvez :
1. Jouer au Kahoot
2. Télécharger le rapport
3. Utiliser l'onglet "RawReportData" du fichier Excel qui contient questions et réponses

## Limitations de Kahoot! à connaître

- ❌ **Pas de feedbacks pédagogiques** : Impossible d'inclure des explications détaillées dans l'import
- ❌ **Pas de formules complexes** : Équations mathématiques limitées
- ⚠️ **4 réponses maximum** (plans gratuits) ou 6 (plans payants)
- ⚠️ **Contraintes de longueur strictes** (d'où l'importance du format QCM léger)

Pour des QCM nécessitant feedbacks détaillés, contextualisation, ou questions plus longues, privilégiez les formats Moodle (GIFT/XML) ou Word.
