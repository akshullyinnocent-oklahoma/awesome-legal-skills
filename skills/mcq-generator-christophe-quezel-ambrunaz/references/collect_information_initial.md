# Création de QCM - Collecte des informations

Je vais vous accompagner dans la création d'un QCM de qualité. Pour ce faire, j'ai besoin de recueillir plusieurs informations essentielles qui détermineront la structure et le contenu de votre questionnaire.

## ⚠️ PREMIÈRE QUESTION PRIORITAIRE : Type d'usage

**Quel type de QCM souhaitez-vous créer ?**

### Option A : QCM Léger (ludique, révision)

**Caractéristiques** :
- Questions courtes et percutantes (max 95 caractères)
- Réponses concises (max 60 caractères)
- Pas de feedbacks pédagogiques détaillés
- Format adapté à Kahoot!, Wooclap pour usage interactif

**Cas d'usage** :
- Quiz de révision en classe
- Activation cognitive en début de séance
- Évaluation formative ludique
- Entraînement répété de connaissances factuelles

### Option B : QCM Approfondi (évaluatif, formatif)

**Caractéristiques** :
- Questions de longueur libre avec contextualisation possible
- Réponses détaillées
- Feedbacks pédagogiques complets pour chaque option
- Format adapté à Moodle, examens papier Word

**Cas d'usage** :
- Évaluation sommative (examens, contrôles)
- Évaluation formative avec apprentissage intégré
- Test de compréhension conceptuelle
- Diagnostic pédagogique précis

## Informations requises

Une fois le type choisi, pourriez-vous me préciser les éléments suivants :

### 1. Paramètres quantitatifs
- **Nombre de questions** à générer
- **Nombre de réponses possibles** par question (je recommande 4 à 6 options pour éviter la surcharge cognitive)

### 2. Type de réponse
- **Choix unique** : Une seule réponse correcte par question
- **Réponses multiples** : Plusieurs réponses correctes possibles, avec pondération

### 3. Système de notation et pénalités
- **Souhaitez-vous des points négatifs pour les réponses incorrectes ?**
  - Si oui : quelle pénalité ? (exemples courants : -0.25, -0.33, -0.5, -1 point par erreur)
  - Si non : les réponses incorrectes rapportent 0 point
  
*Justification pédagogique* : Les points négatifs découragent les réponses au hasard et valorisent la prudence, mais peuvent aussi pénaliser des étudiants qui hésitent légitimement. Le choix dépend de vos objectifs pédagogiques.

### 4. Contenu thématique
- **Thématique/sujet précis** du QCM
- **Sous-thèmes spécifiques** à couvrir (si applicable)

### 5. Niveau du public visé
Veuillez indiquer le niveau visé parmi :
- **Primaire** : Élèves de cycle 1, 2, ou 3 (CP-CM2)
- **Collège** : Élèves de 6ème à 3ème
- **Lycée** : Élèves de Seconde à Terminale
- **Licence** : Étudiants de L1 à L3 (vous pouvez préciser : "Licence 1", "Licence 3", etc.)
- **Master** : Étudiants de M1 à M2 (vous pouvez préciser : "Master 2 recherche", etc.)

*Note* : Si votre public ne correspond pas exactement à ces catégories (formation continue, doctorat, etc.), précisez-le en texte libre et j'adapterai le niveau de complexité en conséquence.

### 6. Objectif pédagogique
Quel est le but principal de ce QCM ?
- Évaluation sommative (examen, contrôle certificatif)
- Évaluation formative (auto-évaluation, entraînement avec apprentissage)
- Vérification de connaissances préalables
- Stimulation de la réflexion et de l'analyse critique

### 7. Documents de référence (fortement recommandé)
Disposez-vous de documents de référence que vous souhaiteriez me fournir pour alimenter le contenu du QCM ?
- Cours, polycopiés
- Articles scientifiques
- Manuels, ouvrages de référence
- Supports de présentation

Si oui, vous pouvez les charger dès maintenant. Ces documents me permettront de générer des questions parfaitement alignées avec votre enseignement.
