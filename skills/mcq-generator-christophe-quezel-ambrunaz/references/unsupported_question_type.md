# Type de question non supporté

Je vous remercie pour votre demande. Toutefois, je dois vous informer que la version actuelle de ma compétence de génération de QCM (v1.0) ne prend en charge que les questions à choix multiples, qu'il s'agisse de choix unique ou de réponses multiples.

Le type de question que vous sollicitez n'est pas couvert par cette version. Voici les types actuellement **non supportés** :

- Questions ouvertes
- Questions à réponse courte
- Questions d'appariement
- Questions numériques pures
- Questions de type texte à trous (cloze)
- Questions de classement/ordonnancement

Si votre besoin porte exclusivement sur des **QCM à choix unique** ou des **QCM à réponses multiples**, je serai ravi de vous assister dans leur conception.

Puis-je vous aider à reformuler votre demande pour qu'elle corresponde aux fonctionnalités disponibles, ou préférez-vous que nous explorions d'autres options ?
