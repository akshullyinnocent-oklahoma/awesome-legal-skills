# Current Official Source Map

Use this map to pick starting points by workflow. It intentionally avoids freezing revision-specific download URLs.

## HTS / Classification

| Need | Preferred source |
|---|---|
| Current HTS bulk JSON | Data.gov catalog record for "Harmonized Tariff Schedule of the United States ({year})" |
| Live tariff-line search | USITC HTS REST search at `https://hts.usitc.gov/reststop/search?keyword={term}` |
| Chapter and section notes | USITC HTS current release / chapter files |
| Historical revisions | USITC HTS Archive |
| HTS learning/context | USITC HTS information and official HTS guide |

## CBP / CROSS / Customs Guidance

| Need | Preferred source |
|---|---|
| Binding ruling text | CBP CROSS ruling page at `https://rulings.cbp.gov/ruling/{id}` |
| Ruling search | CBP CROSS search |
| Ruling revocations/modifications | CROSS ruling text, CBP Customs Bulletin when relevant |
| Informed Compliance Publications | CBP official publications pages |
| MPF and entry fee guidance | CBP official fee pages and 19 CFR Part 24 |

## Courts

| Need | Preferred source |
|---|---|
| CIT slip opinions | `https://www.cit.uscourts.gov/content/slip-opinions-{year}` |
| CIT opinion text | Official CIT PDF |
| CAFC opinions | Official Federal Circuit opinions and orders pages |
| Subsequent history | Official appellate docket/opinion source when available; use secondary citators only as orientation if no official source is available |

## Trade Remedies

| Need | Preferred source |
|---|---|
| Section 301 | USTR official notices, product lists, exclusions, Federal Register notices, HTS Chapter 99 |
| Section 232 | Presidential proclamations, Commerce/BIS exclusion materials, Federal Register, HTS Chapter 99 |
| Section 201 | Presidential proclamations, USITC safeguard materials, Federal Register, HTS Chapter 99 |
| AD/CVD orders and scope | Commerce/ITA, ACCESS where available, Federal Register notices, CBP liquidation/cash deposit instructions when available |

## Origin / Preference Programs / Procurement

| Need | Preferred source |
|---|---|
| Marking and substantial transformation | 19 U.S.C. 1304, 19 CFR Part 134, CBP CROSS origin rulings |
| USMCA and FTA rules | Official agreement text, CBP/USMCA guidance, USTR/CBP official sources |
| Special program codes | Current HTS special field plus `references/fta-program-codes.json` as a decoder |
| TAA designated countries | FAR/DFARS or GSA official sources, current-year procurement guidance |

## PGA / Admissibility

| Need | Preferred source |
|---|---|
| FDA | FDA import program, product-specific import pages, FDA product codes where relevant |
| EPA | EPA import requirements, TSCA, pesticides, vehicles/engines, ozone-depleting substances |
| CPSC | CPSC regulated products, certificates, testing and labeling requirements |
| FCC | FCC equipment authorization and radiofrequency device import guidance |
| USDA/APHIS | APHIS import requirements, plants, animals, agricultural products |
| DOT/PHMSA/NHTSA | Official vehicle and hazardous materials import rules |
| FWS/NMFS | Wildlife, fishery, and protected species import requirements |

## UFLPA / Forced Labor

| Need | Preferred source |
|---|---|
| UFLPA operational guidance | CBP UFLPA official guidance |
| Entity List | DHS UFLPA Entity List official materials |
| Withhold Release Orders | CBP WRO and forced labor pages |
| Risk sectors | Official DHS/CBP strategy and enforcement updates |

## Secondary Sources

Use law firm alerts, trade press, treatises, or database snippets only to orient research or identify official sources. Label them as secondary and do not use them as controlling authority.
